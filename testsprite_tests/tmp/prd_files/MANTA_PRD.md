# Product Requirements Document (PRD) - Manta Studio

## 1. Project Overview
Manta Studio is a premium digital atelier specializing in branding, design, and high-end digital experiences. The landing page serves as a showcase of their philosophy: merging organic gestures with precise digital structure.

## 2. Core Features & Navigation
- **Hero Section**: High-impact visual introduction with interactive elements.
- **Brand Intro**: Parallax-driven section introducing the "Atelier Digital" concept.
- **Manifesto & Services**: Detailed explanation of the studio's approach and service offerings (Branding, Web Design, Strategy).
- **Portfolio Section**: A Bento Grid layout showcasing real projects with micro-interactions.
- **Lab Section**: An experimental area for creative coding and technical demos.
- **Contact Section**: A dynamic, premium form for project inquiries with social media integration.
- **NotFound (404)**: A "Blank Canvas" wavy experience designed to keep users engaged even when they get lost.

## 3. Visual Identity & UX
- **Design System**: "Manta Ink" (#D9026D), "Paper Light" (#F4F1EA), and "Deep Charcoal" (#1A1A1A).
- **Textures**: Noise overlays and sketch-grid backgrounds for a tactile, physical feel.
- **Interactions**: Custom cursor, GSAP animations, Framer Motion transitions, and Three.js (3D) components.

## 4. Technical Stack
- **Frontend**: React 19, TypeScript, Vite.
- **Styling**: Tailwind CSS 4.0.
- **Animation**: Framer Motion, GSAP.
- **3D**: React Three Fiber (R3F), Three.js.
- **Analytics**: Umami Analytics.

## 5. Objectives
- High conversion for project inquiries.
- Flawless performance and SEO optimization.
- "World Class" mobile and desktop responsiveness.
